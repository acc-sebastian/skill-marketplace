Add a skill or prompt

Once merged to main it appears here within minutes.

  1. Fork the GitHub repo
  2. Add metadata.json + SKILL.md under skills/ — or metadata.json + PROMPT.md under prompts/
  3. Open a pull request

No Git? Use the form instead — it opens a pre-filled issue and scaffolds the skill for you.

Manual install & other harnesses

Claude Code (single skill): put SKILL.md into .claude/skills/<skill-id>/ (project) or ~/.claude/skills/<skill-id>/ (global), then restart or run /reload.

Copilot Studio: copy the content after the frontmatter (below the second ---) into the System Prompt of a Topic, then publish.

Any LLM: paste the content after the frontmatter as a system prompt or first message.
